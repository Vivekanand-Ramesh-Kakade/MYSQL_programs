# MYSQL_programs
Programs featuring the concept of the database with the MYSQL programs

MYSQL is an SQL program used to perform various operations with the databases

Types of the database
- Relational database
- Non-relational database

Types of database storage
- Onsite
- Cloud

Relational database are the database which stores the data a tabular form.
The database consist different tables/schemas
The tables consist of row and columns
Rows are called as records
Columns are called as field/Attributes

Operations performed on database are classified as CRUD
- Create
- Retrive/ Read
- Update/ Modify
- Delete

A database admin concepts are
- Application installation and configurations
- database/tables creations'
- User access grant and revoke
- data security/ governance
- performance monitoring
- backups and recovery
- disater management
- High availablity and failover
- indexing and optimising queries
